## 一、战略转型背景

本次升级的核心目标是将 TokenStar 网站从"**内容聚合平台**"转型为"**企业智能实施平台**"。

原有网站以资讯、知识库、Skills 市场等内容型模块为核心，适合开发者和技术爱好者浏览，但对于企业决策者（A 级用户）的转化路径不清晰，缺乏明确的价值主张和行动引导。

**升级后的核心定位：**

> TokenStar 是 OpenClaw 中国生态的企业级实施与落地平台，专注于帮助企业决策者理解、评估并落地 AI Agent 解决方案。

## 二、导航结构重构

### 2.1 新版一级导航（7 个入口）

| 导航项 | 目标 URL | 战略定位 | 目标用户 |
|--------|----------|----------|----------|
| 首页 | `/` | 企业价值主张入口 | 所有用户 |
| 企业解决方案 | `/scenarios` | 核心转化入口 | 企业决策者（A 级） |
| 成功案例 | `/cases` | 信任建立 + 转化 | 企业决策者（A 级） |
| 部署方案 | `/deployment` | 购买决策支持 | IT 负责人（B 级） |
| 企业培训 | `/training` | 服务深化 | 企业管理者 |
| 资源中心 | `/resources` | 内容聚合 | 开发者 + 技术用户 |
| 关于我们 | `/about` | 品牌信任 | 所有用户 |

### 2.2 导航设计原则

- **企业解决方案**置于首位，突出平台核心价值
- **成功案例**独立为一级导航，增强信任背书
- 所有内容型页面（新闻、Blog、Skills、知识库、指南）收归**资源中心**，减少导航噪音
- 保留所有原有 URL，不破坏 SEO

### 2.3 资源中心子菜单

```
资源中心 (/resources)
├── OpenClaw 专区 (/openclaw)
├── 生态动态 (/ecosystem)
├── Skills 市场 (/skills)
├── AI 管理指南 (/guide)
├── 新闻中心 (/news)
├── Blog (/blog)
└── 知识库 (/knowledge)
```

## 三、首页模块重排

### 3.1 新版首页模块顺序

| 顺序 | 模块名称 | 核心目的 |
|------|----------|----------|
| 1 | Hero — 企业价值主张 | 第一屏建立认知，双 CTA 引导 |
| 2 | 企业智能体解决方案 | 6 大场景入口，引导至 /scenarios |
| 3 | 企业落地案例精选 | 信任建立，引导至 /cases |
| 4 | 三种部署路径 | 购买决策支持，引导至 /deployment |
| 5 | 企业培训支持 | 服务深化，引导至 /training |
| 6 | 资源中心入口 | 内容消费，引导至 /resources |
| 7 | 收口 CTA | 预约评估 + 获取方案 |

### 3.2 设计要点

- Hero 标题聚焦企业价值："让 AI 成为每个部门的超级管理 Agent"
- 统计数字强化可信度：8+ 行业、24+ 案例、6 个场景
- 案例精选直接展示 `mockCases.featured` 数据，无数据时降级为行业覆盖摘要
- 资源中心以 6 宫格快速入口呈现，PC 6 列 / 移动端 2 列

- **Hero 区**：标题 + 快速链接标签云

- **底部 CTA**：引导至 /scenarios 和 /cases

## 六、转化漏斗逻辑

本次升级强化了以下转化路径：

### 路径 A：企业决策者（A 级用户）
```
首页 Hero → /scenarios（解决方案）→ /cases（案例验证）→ /deployment（部署评估）→ /about#contact（预约）
```

### 路径 B：技术评估者（B 级用户）
```
首页 → /deployment（部署方案）→ ROI 测算 → /cases（案例参考）→ /about#contact（联系）
```

### 路径 C：内容消费者（C 级用户）
```
首页资源中心入口 → /resources → 各内容页 → 案例/解决方案 CTA → 转化
```

## 七、未来扩展建议

### 7.1 短期（1-3 个月）

1. **行业专题页**：为每个行业（电商、制造、金融等）创建独立落地页，提升 SEO 长尾词覆盖
2. **案例详情页增强**：在 `/cases/[slug]` 中增加"相关解决方案"和"部署路径"推荐模块
3. **ROI 计算器独立入口**：将 ROI 测算作为独立工具页面，增加 SEO 流量入口

### 7.2 中期（3-6 个月）

1. **企业自助评估工具**：基于行业 + 规模 + 场景的智能推荐系统
2. **客户故事视频**：在案例页增加视频见证，提升信任度
3. **合作伙伴生态页**：展示 OpenClaw 生态合作伙伴，增强平台可信度

### 7.3 长期（6-12 个月）

1. **企业会员中心**：注册企业用户可获取定制化内容和专属资源
2. **在线演示预约系统**：集成日历预约，降低销售转化摩擦
3. **多语言支持**：英文版本，覆盖海外华人企业市场
